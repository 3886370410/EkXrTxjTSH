# 项目简介

本系统名为基于深度学习YOLOv12的手机吸烟水杯检测系统。该系统通过整合YOLOv12算法与自定义YOLO数据集，实现了对手机拍摄画面中吸烟水杯的实时检测与识别。系统包含UI界面，提供了直观的操作流程，同时设有登录注册界面，确保用户数据的安全性。技术特点方面，系统采用了深度学习技术，具有较高的检测准确率和实时性，为相关领域的智能化管理提供了有效的技术支持。项目代码的开源性质也便于同行交流与技术研究。


## 核心功能
- 基于YOLOv12算法实现对手机拍摄画面中吸烟水杯的实时检测与识别
- 整合自定义YOLO数据集提高检测准确性
- 提供直观的UI界面简化操作流程
- 配备登录注册界面保障用户数据安全
- 提供完整的项目源代码支持技术研究与交流

## 适用场景
- 手机拍摄内容的智能分析
- 特定物品（吸烟水杯）的实时检测
- 需要高准确率与实时性的检测场景
- 智能化管理系统的技术支持
- 深度学习技术的研究与应用

## 技术特点
- 采用深度学习技术实现高准确率检测
- 具备良好的实时性能
- 开源性质便于技术交流与二次开发
- 模块化设计便于功能扩展
